Politique de Confidentialité

Protection de vos données personnelles

1. Introduction

Charlie's Events (ci-après « nous », « notre » ou « la Société ») s'engage à protéger votre vie privée. Cette Politique de Confidentialité explique comment nous collectons, utilisons, divulguons et sauvegardons vos données personnelles.

2. Données Collectées

Nous collectons les données personnelles que vous nous fournissez volontairement :

  • Via le formulaire de contact : nom, adresse email, téléphone, entreprise, secteur d'activité, type d'événement, budget, message
  • Via les cookies : informations sur votre navigation et vos préférences
  • Données de connexion : adresse IP, type de navigateur, pages consultées, durée de visite

3. Utilisation des Données

Nous utilisons vos données personnelles pour :

  • Répondre à vos demandes de renseignements et offres de services
  • Envoyer des communications marketing (si vous y avez consenti)
  • Améliorer notre site web et nos services
  • Analyser l'utilisation du site et optimiser l'expérience utilisateur
  • Respecter nos obligations légales

4. Base Légale du Traitement

Le traitement de vos données personnelles est basé sur :

  • Votre consentement : pour les communications marketing
  • L'exécution d'un contrat : pour la fourniture de nos services
  • Nos intérêts légitimes : pour analyser et améliorer nos services
  • Les obligations légales : pour respecter la législation

5. Partage des Données

Nous ne partageons vos données personnelles qu'avec :

  • Les tiers nécessaires à l'exécution de nos services (prestataires, partenaires événementiques)
  • Les autorités compétentes, si la loi l'exige

Nous ne vendons jamais vos données personnelles à des tiers.

6. Durée de Rétention des Données

Vos données personnelles sont conservées pendant :

  • Formulaire de contact : 3 ans après la dernière interaction
  • Données de navigation (cookies) : jusqu'à 1 an maximum
  • Données clients : pendant la relation commerciale + 3 ans après

7. Cookies

Notre site utilise des cookies pour :

  • Cookies essentiels : fonctionnement du site (obligatoires)
  • Cookies analytiques : analyser votre utilisation du site
  • Cookies marketing : personnaliser vos expériences (si consentement donné)

Vous pouvez désactiver les cookies via les paramètres de votre navigateur.

8. Vos Droits

Vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger vos données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la limitation : limiter le traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format portable
  • Droit d'opposition : vous opposer au traitement marketing

9. Sécurité des Données

Nous mettons en place des mesures de sécurité appropriées pour protéger vos données contre l'accès non autorisé, la modification ou la destruction.

10. Transferts Internationaux

Certaines de vos données peuvent être transférées hors de l'Union Européenne. Ces transferts sont effectués en conformité avec les lois applicables (notamment le RGPD).

11. Modifications de cette Politique

Nous nous réservons le droit de modifier cette Politique de Confidentialité. Les modifications entrent en vigueur dès leur publication sur le site.
